# Ejercicio_practico_II
Como un equipo de programadores, colaborarán simultanea para construir una lista de tareas/pendientes en una aplicación web que solo funcione del lado del cliente (solo fronted). Podrás utilizar HTML, Javascript y CSS para elaborar la aplicación, puedes apoyarte de IA para agilizar el proceso de codificación, solo procura evitar el aspecto genérico
